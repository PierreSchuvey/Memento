Lignes de commandes

La ligne de commande

n.b : Précéder la commande de sudo pour l'éffectuer en tant que Super utilisateur

Affichage des contenus

Pour afficher le manuel d'une commande j'utilise : man (nom de la commande)

Pour afficher le contenus d'un dossier j'utilise : ls

Pour afficher la version longue du contenus d'un fichier j'utilise : ls - l

Pour afficher le contenus d'un fichier en plus des fichiers cachés j'utilise : ls -a

Pour afficher le contenus d'un fichier en version longue plus les fichiers cachés j'utilise : ls -l -a

Pour afficher les dossier parents j'utilise : pwd

Pour me rendre dans un dossier j'utilise : cd ( le chemin du dossier désiré)

Gestion de paquets

Pour mettre à jour la liste des paquets j'utilise : apt update

Pour mettre à jour les paquets j'utilise : apt upgrade

Pour rechercher un paquet j'utilise : apt search (nom du paqet)

Pour installer un paquet j'utilise : apt install (nom du paqet)

Gestion de documents

Pour créer un dossier je dois utiliser cette commande : mkdir

Pour créer un dossier et des sous dossier j'utilise mkdir p (hierarchie)

Pour créer un fichier j'utilise touch

Pour supprimer un dossier j'utilise rm

Pour déplacer un dossier j'utilise mv
